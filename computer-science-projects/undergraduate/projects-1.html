Arrays | Resizable Bags


Programming Language : Java

Use Arrays to Manipulate Resizable Bags

This program shows the use of Arrays and Linked Lists to generate Bags. Similar to normal bags, these contain values of any type. However, this type of bag allows the size to be dynamic.

The initial set size of the array (Bag) is not set in stone, and changes upon the limit being reached. When the Bag is full, and a new item is to be added, the Bag size will double (copy over all values), and allow for additional entries.

These Bags can contain any data type (integers, strings, char...).
